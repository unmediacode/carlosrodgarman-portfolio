# Technical Videos - Images Guide

Este folder contiene las thumbnails (miniaturas) de los videos técnicos en la subcarpeta **miniaturas/**.

## Ubicación de las Imágenes

Todas las miniaturas deben estar en: `technical-videos/images/miniaturas/`

## Especificaciones de Imágenes

### Formato y Dimensiones
- **Aspecto:** 16:9 (1920x1080px recomendado)
- **Formato:** JPG optimizado
- **Peso máximo:** 200KB por imagen
- **Calidad:** Alta resolución para visualización en web

## Sugerencias

- Usar capturas de los propios videos de YouTube como thumbnails
- Asegurarse de que el texto sea legible en las imágenes
- Mantener un estilo visual consistente entre todas las thumbnails
- Si no tienes una imagen específica, se usará el placeholder.svg automáticamente

## Placeholder

El archivo `placeholder.svg` se usa automáticamente cuando falta una imagen específica.
