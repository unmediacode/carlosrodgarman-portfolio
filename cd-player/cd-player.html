CD Player Component

Interactive album player with expandable panels and streaming links

How to Use

1. Include the CSS and JS files:

<link rel="stylesheet" href="cd-player/cd-player.css">
<script src="cd-player/cd-player.js"></script>

2. Create a container element:

<div class="cd-player" data-album-id="impossible-dream"></div>

3. Initialize the players:

<script>
    // Initialize all CD players with data from JSON
    initCDPlayers('cd-player/data.json');
</script>

4. Update the JSON file with your album data:

Edit cd-player/data.json to add your albums. Each album needs:

  • id: Unique identifier
  • coverImage: Path to album cover image
  • artist: Artist name
  • title: Album title
  • year: Release year
  • completedWork: Your role (Engineer, Producer, etc.)
  • audioFile: Path to preview audio file
  • links: Object with appleMusic, spotify, and amazon URLs

Features

  • Hover Effect: Play and Info buttons appear on hover
  • Play Button: Click to play/pause audio. Shows "Completed work" panel when playing
  • Info Button (+): Opens modal with full album details, volume control, and streaming links
  • Responsive: Works perfectly on desktop, tablet, and mobile
  • Multiple Players: Add as many players as you need on the same page
  • Data-driven: All content loaded from JSON file
